# tagus_scoreboard
A scoreboard to FIVEM in QBCore Framework using `ox_lib`

## New Features (Added/Updated)
- Converted from ESX to QBCore
- Easily add jobs in the `server.lua`
- Adjust sorting order how you want to view it in `ox_lib`
- Shows group jobs (ex. mechanics) in one category (optional)
- Shows Citizen ID
- Hides specific jobs (ex. fib only shows to police and sheriff, but not to civilians)
- On duty to see the specific job(s)

## How To Use
- Press `F1` or type `/scoreboards`

## Installation
- Drag into your `resources` folder
- ensure `tagus_scoreboard` in your server.cfg
- In-game, press `F1` or type `/scoreboards`

## Dependencies
- QBCore https://github.com/qbcore-framework/qb-core
- ox_lib https://github.com/overextended/ox_lib
